)
Lavaca.mvc.Collection

Defined in src/www/components/lavaca/src/mvc/Collection.js:545

Sorts the models in the collection according to the specified compare function.

Parameters:

  • compareFunction Function

    A function that compares two models. It should work in the same manner as the default Array.sort method in javascript. i.e. the function should have a signature of function(modelA, modelB) and should return a negative integer if modelA should come before modelB, a positive integer if modelB should come before modelA and integer 0 if modelA and modelB are equivalent.

Returns:

Lavaca.mvc.Collection: The updated collection (for chaining)
